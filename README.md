# **Automated Office Workflows:**

OfficeAutos is a suite of tools designed to streamline and automate common administrative and data-handling tasks within an office environment. The project aims to improve efficiency by providing desktop applications built with JavaScript, and python for the programming.

The core functionality includes file manipulation, data processing, and workflow orchestration, driven by the modular filemovers  and DocX generator template sub-projects.
